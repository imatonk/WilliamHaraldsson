## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Customization

Replace the example content in `app/page.tsx` with your own information:

- **Hero Section**: Update your name, title, and tagline
- **About Section**: Write your own bio (3-4 sentences)
- **Skills Section**: List your technical skills
- **Experience Section**: Add your work experience
- **Projects Section**: Showcase your projects with tech stacks
- **Contact Section**: Update email, GitHub, and LinkedIn links

Also update the metadata in `app/layout.tsx` for SEO.

## Build for Production

```bash
npm run build
```

## Deploy to Vercel

1. Push your code to GitHub
2. Import your repository on [Vercel](https://vercel.com)
3. Deploy with one click
